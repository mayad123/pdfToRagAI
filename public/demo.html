Try demo (MCP over HTTP)

Requires npm run mcp:http running on this machine. The page loads the MCP SDK from esm.sh (network on first use) and calls tools on /mcp. Paths must satisfy your server’s PDF_TO_RAG_ALLOWED_DIRS / PDF_TO_RAG_CWD policy.

MCP from this server

Same tools as Cursor stdio: ingest, query, inspect. Status updates when the browser session connects.

MCP session: not connected

Commands to start this server from the project root:

npm run build
npm run mcp:http
# → UI http://127.0.0.1:3000/  ·  MCP http://127.0.0.1:3000/mcp

Example mcp.json snippet (replace paths). See Setup for steps.

{
  "mcpServers": {
    "pdf-to-rag": {
      "command": "node",
      "args": ["/absolute/path/to/pdfToRag/dist/mcp/server.js"],
      "env": {
        "PDF_TO_RAG_CWD": "/absolute/path/to/pdfToRag",
        "PDF_TO_RAG_SOURCE_DIR": "/absolute/path/to/pdfToRag/examples",
        "PDF_TO_RAG_ALLOWED_DIRS": "/absolute/path/to/pdfToRag/examples"
      }
    }
  }
}
