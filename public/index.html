pdf-to-rag · browser demo (MCP HTTP)

Served by pdf-to-rag-mcp-http. The page loads the official MCP TypeScript client from esm.sh (needs network on first use) and calls tools on /mcp (same origin — no CORS issues). Use paths on the server machine that satisfy your PDF_TO_RAG_ALLOWED_DIRS / PDF_TO_RAG_CWD policy.

Default matches a dedicated folder under the server process working directory.
