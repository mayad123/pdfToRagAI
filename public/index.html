Local · Private · Cited answers

Search your PDFs without sending them to the cloud

Point the tool at a folder of PDFs, build an index on your computer, then ask questions in plain English. Every answer includes which file and which page the text came from—so you can verify and cite sources.

How it works

1

Choose your folder

Keep research papers, manuals, or reports in one directory on your machine.

2

Index once

The app reads the PDFs, chunks the text, and stores embeddings locally for fast search.

3

Ask naturally

Type a question; get back short excerpts with filenames and page numbers—not a black-box summary.

Set up on your computer Try the live demo

The live demo only works after you start the app from a terminal (npm run mcp:http). That same command serves this website and connects the buttons to your local index—nothing runs in the browser alone.
